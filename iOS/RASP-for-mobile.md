# Runtime application self-protection

## Threatens
  - compromised device
  - lost or stolen device
  - malicious app
  - repackage of app

## Features
  - jailbreak
  - emulator
  - repackaging
  - runtime lib injection
  - system screenshot
  - screen mirroring
  - native code hook
  - xcode/java debugger

## Harden Mechanism
  - config and integrate RASP
  - RASP SDK wrapping & binding
  - Application re-signing


## Approaches
  - detection
  - blocking
  - app termination
